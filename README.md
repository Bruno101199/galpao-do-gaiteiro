# Galpão do Gaiteiro — Landing Page

Site de apresentação e captação de agendamentos para o **Galpão do Gaiteiro**, salão de festas em Alvorada/RS.

## Sobre o Projeto

Landing page responsiva criada para gerar agendamentos de eventos no Galpão do Gaiteiro — um espaço com ambiente gaúcho autêntico, capacidade para 120 pessoas e pátio gramado, localizado em Alvorada, Rio Grande do Sul.

## Seções

- **Hero** — Foto da fachada com chamada principal e botão de orçamento via WhatsApp
- **Barra de Benefícios** — Diferenciais do espaço (capacidade, pátio gramado, estacionamento, ambiente gaúcho)
- **Eventos** — Tipos de eventos realizados: Festa de 15 Anos, Aniversários e Confraternizações
- **O Espaço** — Fotos e diferenciais do local
- **Galeria** — Carrossel com 6 fotos do espaço e eventos
- **Nossa História** — Apresentação de Inácio Guerreiro com jornada do herói
- **Contato** — Mapa, telefone, endereço e links para redes sociais
- **CTA Final** — Chamada para agendamento via WhatsApp

## Tecnologias

- HTML5 semântico
- CSS3 (variáveis, grid, flexbox, animações)
- JavaScript vanilla (carrossel, scroll reveal, menu mobile)
- Google Fonts (Playfair Display + Lato)
- Google Maps Embed
